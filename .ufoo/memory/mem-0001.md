# Mutable static delegate is intentional

MoLc Mutable<T> intentionally uses a static build-phase delegate to capture the currently building MutableWidget and register automatic subscriptions. This relies on Flutter widget builds running synchronously on one isolate and is meant to provide GetX-like dependency tracking. Do not treat the delegate as accidental global state or replace it with explicit subscriptions unless changing Mutable's core tracking semantics.
